# EventLite.js

[![NPM Version](https://img.shields.io/npm/v/eventlite.js)](https://www.npmjs.com/package/eventlite.js)

EventLite.js is a very simple and fast event emitter, optimized code less than 300 LOC, zero dependencies, written using ES6 standard and works in most JavaScript runtime.
